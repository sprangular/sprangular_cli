# Sprangular Utility

A tool for creating sprangular extensions and setting up apps.

## Installation


```shell
gem install sprangular_cmd
```

## Usage

### Create an extension

```shell
sprangular extension example
```

### Setup an application

```shell
cd my-rails-app
sprangular install
```

### Show available commands

```shell
sprangular help
```

### Show version

```shell
sprangular -v
```
